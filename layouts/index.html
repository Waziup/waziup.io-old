Our technology

WaziDev

WaziDev is a Sensing and Actuation platform for your applications! It can send your data up to 7 Km using the LoRa technology. It is easily programmable and customizable, using Arduino technology.

WaziGate

WaziGate is a LoRa gateway that can receive and emit LoRa messages up to 200 devices! It supports LoRaWAN, edge applications and actuation. You can control is from remote using the WaziCloud.

WaziCloud

WaziCloud allows Remote control of your application! Using the dashboard, you can visualize you data and control you devices and gateways. It offers an Open API for developping your IoT applications.
